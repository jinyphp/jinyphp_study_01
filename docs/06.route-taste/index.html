🍯 라우팅 맛보기

목표: 5분 안에 라우팅이 뭔지 알고 첫 번째 라우트 만들어보기!

📚 학습 모듈

라우팅을 체계적으로 이해하기 위해 3단계로 나누어 학습합니다:

Module 1: 주소란?

  • 🏠 웹 주소(URL) 체계 이해하기
  • 🗺️ URL 구조 완전 분석
  • 📍 실생활 비유로 주소 개념 익히기
  • 🎯 좋은 URL 설계 원칙

Module 2: 라우트의 필요성

  • 🤔 왜 라우팅 시스템이 필요할까?
  • 😰 라우팅 없는 옛날 웹사이트의 문제점
  • 📈 라우팅 도입 후 놀라운 장점들
  • 🌍 실제 성공 사례들

Module 3: 라우트의 동작

  • ⚙️ Laravel 라우팅 작동 원리
  • 📝 HTTP 메소드 완전 이해
  • 🎨 다양한 응답 방식 마스터
  • 🛠️ 실습으로 직접 만들어보기

🎯 빠른 시작 (2분 체험!)

바로 체험해보고 싶다면, 가장 간단한 라우트부터 시작해보세요:

🚀 첫 번째 라우트 만들기 (2분 완성)

1단계: routes/web.php 파일에 추가하세요:

Route::get('/hello', function () {
    return '안녕하세요! 첫 번째 라우트입니다!';
});

2단계: 브라우저에서 http://localhost:8000/hello 접속

결과: “안녕하세요! 첫 번째 라우트입니다!” 화면에 나타남! 🎉

🎯 간단 체험 미션 (3분)

// 현재 시간 보여주기
Route::get('/time', function () {
    return '현재 시간: ' . date('Y-m-d H:i:s');
});

// 예쁜 HTML 페이지
Route::get('/welcome', function () {
    return '<h1>🎉 환영합니다!</h1>
            <p>Laravel 라우팅 세계에 오신 것을 환영합니다!</p>';
});

확인: /time, /welcome 페이지 접속해보세요!

🏆 축하합니다!

🎉 라우팅 첫걸음 성공!

이제 여러분은:

  • ✅ 라우트를 만들 수 있어요
  • ✅ 브라우저에서 확인할 수 있어요

📖 체계적으로 배우기

라우팅을 완전히 이해하고 싶다면 위의 학습 모듈을 순서대로 학습하세요:

  1. 주소란? → URL 구조 이해 (5분)
  2. 라우트의 필요성 → 왜 필요한가? (7분)
  3. 라우트의 동작 → 실제 작동 원리 (10분)
